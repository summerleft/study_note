### ```display: none```
+ 触发页面的重绘和重排（回流）
+ 不占据空间
+ 不触发绑定事件
+ 在DOM结构上可见

### ```visibility: hidden```
+ 触发页面的重绘
+ 占据空间
+ 不触发绑定事件

### ```opacity: 0```
+ 设置元素的不透明度，0表示全透明
+ 触发页面的重绘
+ 占据空间
+ 触发绑定事件

### ```transform: scale(0)```
+ scale代表缩放，1为不缩放，0表示缩放为最小
+ 触发页面的重绘
+ 占据空间
+ 不触发绑定事件

### width和height设置为0
+ 如果它有子元素，需要设置```overflow: hidden```
+ 触发页面的重绘和重排
+ 不占据空间
+ 不触发事件
